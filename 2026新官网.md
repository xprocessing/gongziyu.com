### 一、首次拉取（克隆）仓库到本地
如果本地还没有这个仓库，先执行**克隆**操作：
1. 打开终端/命令行（Windows 用 CMD/PowerShell，macOS/Linux 用 Terminal）；
2. 进入你想存放仓库的目录（比如 `cd /Users/xxx/Documents` 或 `cd D:\code`）；
3. 执行克隆命令：
```bash
git clone https://github.com/xprocessing/gongziyu.com.git
```
4. 克隆完成后，进入仓库目录：
```bash
cd gongziyu.com
```

### 二、更新本地仓库（拉取远程最新代码）
如果本地已经克隆过该仓库，需要同步远程最新代码时，执行以下步骤：
1. 先进入仓库根目录（确保当前目录是 `gongziyu.com`）；
2. 执行**拉取**命令（默认拉取 `main`/`master` 分支，根据仓库默认分支调整）：
```bash
git pull origin main
# 如果仓库默认分支是 master，执行：git pull origin master
```
